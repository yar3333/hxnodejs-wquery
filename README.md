# hxnodejs-wquery #

Externals for [wquery](https://github.com/yar3333/haxe-wquery) - a client-only component-oriented web framework.

```shell
# install npm package
npm install hxnodejs-wquery

# install haxelib
haxelib install hxnodejs-wquery

# build.hxml
--library hxnodejs-wquery
```
